## About the Challenge

http://103.167.136.89:10088/

-kemudian ganti value dengan angka 1/n
-selanjutnya refresh browser/n
-di website akan muncul flagnya yaitu: ForestyHC{here_is_your_fortun3_cookie_4a0a47}/n
